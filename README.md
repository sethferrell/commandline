# commandline
First Command - Sensitivity
Input:
~/projects/minigrep$ cargo run to poem.txt
                         
Output:
Are you nobody, too?
How dreary to be somebody!

Second Command - Insensitivity
Input:
~/projects/minigrep$ cargo run to poem.txt -i
                                    
Output:
Are you nobody, too?
How dreary to be somebody!
To tell your name the livelong day
To an admiring bog!
